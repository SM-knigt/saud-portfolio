# Saud Bin Samhan — Portfolio

Personal site showcasing AI and full stack work, shipped projects, and verified programs.

**Live:** [saudb-portfolio.netlify.app](https://saudb-portfolio.netlify.app)

## Stack

- **React** + **Vite**
- **Tailwind CSS**
- **Framer Motion**

## What’s on the site

- Case study style **Projects** (hero imagery, galleries, GitHub + LinkedIn links)
- **Courses & programs** with expandable copy and certificate PDFs
- **Currently** (degree, internship, focus areas)
- **Skills**, **Contact** (email, socials, CV), responsive layout

## Local development

```bash
npm install
npm run dev
```

```bash
npm run build   # output: dist/
npm run preview # serve production build locally
```

Static assets (images, PDFs, `programs/`) live under `public/`.

## Deploy

Connected to **Netlify** from this repo; build settings are in [`netlify.toml`](netlify.toml) (`npm run build`, publish `dist`).
